Web Crypto API TestWebsite für die Browser Extension

Über die folgenden Links kommen Sie auf einzelne Seiten, in der jeweils kryptographische Verfahren inkorrekt umgesetzt wurden.
Die einzelnen kryptographischen Verfahren werden als Regeln definiert, wobei die Browser Extension überprüft, ob diese Regel eingehalten wurde.
Für jede Regel gibt es eine Seite, in der Tests enthalten sind, die das kryptographische Verfahren auf unterschiedliche Weisen inkorrekt umgesetzt haben.
Erst wenn Sie auf den Test Button klicken, wird der Code mit den inkorrekt umgesetzten kryptographischen Verfahren in die Seite geladen.
Auf der jeweiligen Seite wird beschrieben was getestet wird und warum die inkorrekte Umsetzung die Sicherheit mindern kann.



Regel 1: Test auf korrekte nutzung des Initialisierungsvektors

Regel 2: Test auf nutzung einer Signatur nach Nutzung des Verschlüsselungsmodus AES-CBC oder AES-CTR

Regel 3: Test auf Einhaltung der "Encrypt-then-MAC" Regel

Regel 4: Test auf Veränderungen am exportierten Schlüssel

Regel 5: Test auf korrekte nutzung des Salts

Regel 6: Test auf Speicherung des exportierten Schlüssel in den Cookies oder Browser Storage
